RiskSignal is a high-performance, standalone IP intelligence tool designed for security experts. It identifies malicious IPs, detects VPN/Proxy/Tor usage in real-time, and provides anonymity scoring based on multiple signals like hosting status and timezone mismatches.

## Features

- **Real-time IP Analysis**: Instant lookup for any IP address.
- **VPN & Proxy Detection**: Professional-grade detection of anonymizers.
- **Anonymity Scoring**: Unique scoring system that detects browser-side discrepancies (e.g., timezone mismatches).
- **No-Auth & No-Database**: Operates entirely on the frontend with live data integration.
- **Privacy First**: No tracking, no cookies, no account required.

## Technical Details

- **Built With**: React, Vite, TailwindCSS, Lucide Icons, Shadcn/UI.
- **Intelligence Engine**: Live data aggregated from professional IP security feeds.
- **SEO Optimized**: Fully white-labeled with specialized meta tags for visibility.
